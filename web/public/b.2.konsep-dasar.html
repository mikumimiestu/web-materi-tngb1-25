Variabel

Variabel digunakan untuk menyimpan data berupa teks, angka, dan sebagainya yang nialinya dapat diubah-ubah sehingga memudahkan dalam mengakses data tersebut. Dalam penulisan variabel, ada aturan yang harus dipatuhi agar tidak menghasilkan error pada program. Dalam JavaScript, aturan penulisan variabel sebagai berikut:

  1. Jangan menggunakan kata-kata yang menjadi code JavaScript atau dikenal dengan reserved keyword. Contohnya break, bolean, else, and, var, true, enum, switch, this. throw, long, new, dan sebagainya
  2. Variabel tidak boleh di awali dengan angka atau simbol. Variabel harus diawali dengan huruf undersore. Contoh: panjang_tinggi, dan sebagainya. Sedangkan yang tidak dibolehkan seperti &tinggi, 4las, @las, dan sebagainya.
  3. Variabel dalam JavaScript bersifat case-sensitive, artinya hurug kecil dan huruf besar dibedakan. Jadi, antara tinggi dan Tinggi adalah dua variabel yang berbeda.
Variabel pada JavaScript dideklarasikan dengan var. Contohnya sebagai berikut: 
                
                    var nama = "Rizky";
                    var umur = 20;
                    var tinggi = 170;
                    var berat = 60;
Selain dengan cara di atas, dapat juga mendeklarasikan beberapa variabel sekaligus dengan satu var. Misalnya sebagai berikut: 
                
                    var panjang, tinggi;
Variabel juga dapat di deklarasikan dengan langsung mendefenisikan nilai awal. Misalnya sebagai berikut: 
                
                    var panjang = 200;
                    var tinggi = 150;
                    var kata = "JavaScript sangat mudah";
Pada JavaScript terdapat batasan dalam mengakses variabel atau yang disebut scope variabel. JavaScript mengenal variabel global dan variabel local. Variabel global merupakan variabel yang dapat dipakai dimana saja, baik di luar function maupun di dalam function. Sedangakn variabel local yaitu variabel yang hanya dapat digunakan di dalam function. Variabel yang dideklarasikan di luar function akan menjadi variabel global dan variabel yang dideklarasikan di dalam function akan menjadi variabel local. Contohnya sebagai berikut: 
                
                    var a = "Selamat datang";
                    function selamat() {
                        var b = "Selamat pagi";
                        document.write(a)
                        }
                        selamat();
                        document.write(b);
Pada code di atas, variabel a berhasil di tampilkan walaupun dideklarasikan di luar function dan digunakan di dalam function. Ini namanya variabel global. Sedangkan variabel b tidak tampil karena dideklarasikan di dalam function dan digunakan di luar function. Ini namanya variabel local. Variabel local hanya dapat digunakan didalam function di mana variabel tersebut dideklarasikan dan tidak dapat digunakan di luar function atau function lain.
