Pemrograman Web

Link

Link adalah elemen HTML yang digunakan untuk membuat tautan ke halaman web lain. Elemen <a> digunakan untuk membuat link. Elemen ini memiliki atribut href yang digunakan untuk menentukan URL tujuan. Elemen ini juga memiliki atribut target yang digunakan untuk menentukan cara tautan dibuka. Nilai dari atribut target dapat berupa _blank untuk membuka tautan di tab baru, _self untuk membuka tautan di tab yang sama, _parent untuk membuka tautan di parent frame, dan _top untuk membuka tautan di tab paling atas.

Untuk contohnya adalah sebagai berikut:

<a class="text-blue-400 href="https://www.google.com" target="_blank">Google</a>

Hasilnya sebagai berikut:

Google

Dalam contoh di atas, kita dapat melihat struktur dasar HTML yang terdiri dari <a>.


Lalu bagaimana jika kita ingin membuat link yang mengarah ke halaman lain di website kita sendiri? Kita dapat menggunakan href yang mengarah ke path file yang ingin kita tuju.

Untuk contohnya adalah sebagai berikut:

<a href="halman1.html class="text-blue-400">Halman 1</a>

<a href="halman2.html class="text-blue-400">Halaman 2</a>

Hasilnya sebagai berikut:

Halaman 1
Halamn 2

Dalam contoh di atas, kita dapat melihat struktur dasar HTML yang terdiri dari <a>. Dan kita juga dapat melihat bahwa link yang dibuat mengarah ke halaman lain di website kita sendiri.


Lalu bagaimana jika halaman lainnya berada di suatu folder di dalam folder web kita? Kita dapat menggunakan href yang mengarah ke path file yang ingin kita tuju. Jika halaman tersebut berada di dalam folder, kita dapat menambahkan nama folder di depan nama file tersebut. Jika halaman tersebut berada di dalam folder yang berada di dalam folder, kita dapat menambahkan nama folder di depan nama folder tersebut dan nama file tersebut. Begitu seterusnya.

Untuk contohnya adalah sebagai berikut:

<a href="folder1/halman1.html">Halman 1</a>

<a href="folder2/halman2.html">Halaman 2</a>

Hasilnya sebagai berikut:

Halaman 1
Halamn 2

Dalam contoh di atas, kita dapat melihat struktur dasar HTML yang terdiri dari <a>. Dan kita juga dapat melihat bahwa link yang dibuat mengarah ke halaman lain di website kita sendiri yang berada di dalam folder.

Mungkin terlihat sama hasilnya, tapi perbedaan terletak pada path file yang digunakan.
