Pemrograman Web

Pendahuluan

Pemrograman web adalah kegiatan membuat aplikasi web menggunakan bahasa pemrograman tertentu. Bahasa pemrograman yang sering digunakan untuk membuat aplikasi web antara lain HTML, CSS, dan JavaScript. Walaupun sebenarnya html dan css bukanlah bahasa pemrograman. Namun, keduanya sangat penting dalam membuat aplikasi web, selain JavaScript sebagai bahasa pemrograman dalam pembuatan aplikasi web ada juga beberapa bahasa pemrograman lainnya seperti PHP, Python, dan Ruby yang digunakan untuk membuat aplikasi web.

Pada kali ini kita akan mempelajari syntax dasar yang akan menjadi kunci dalam membuat aplikasi web. Namun pada kali ini kita tidak menggunakan (.css) sebagai stylingnya, tetapi kita akan menggunakan tailwind untuk stylingnya.

Mengapa menggunakan tailwind sebagai style?

  • Mempercepat proses styling
  • Mempermudah dalam membuat tampilan
  • Mempermudah dalam membuat tampilan yang responsif


Biasanya untuk html itu di akhir dengan (.html), untuk css di akhiri dengan (.css), dan untuk javascript di akhiri dengan (.js). Namun pada kali ini kita akan menggunakan tailwind sebagai stylingnya, jadi kita akan menggunakan (.css) sebagai file style.

Contoh nama file agar lebih jelas sebagai berikut

  • index.html
  • style.css
  • script.js

Jika menggunakan tailwind maka kita perlu untuk menghubungkan tailwind dengan file html kita dengan cara menambahkan script berikut pada file html kita.

<script src="https://cdn.tailwindcss.com"></script> ini diletakkan pada code html sebelum tag <body> dan setelah tag <title>.

Contoh code html yang sudah terhubung dengan tailwind sebagai berikut

  • <p class="text-sm text-blue-300">Ini adalah tulisan berwarna biru muda menggunakan tailwind</p>
    • maka hasil dari code tersebut adalah

    Ini adalah tulisan berwarna biru muda menggunakan tailwind


Dari contoh penggunaan tailwind di atas, kita bisa melihat bahwa tailwind sangat mempermudah dalam membuat tampilan code yang singkat yang responsif dan mempercepat proses styling.
