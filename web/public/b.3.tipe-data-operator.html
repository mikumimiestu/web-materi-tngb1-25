JavaScript

Tipe Data

Setiap bahasa pemrograman pasti memiliki yang nama tipe data. Tipe data pada JavaScript sedikit berbeda dengan bahasa pemrograman lain. Tipe data pada JavaScript bersifat seolah-olah sebagai objek sehingga setiap variabel pada JavaScript akan langsung memiliki method. JavaScript mengenal 3 tipe data dasar, yaitu sebagai berikut:

  1. Numbers: 125, 20.5, dsb
  2. Strings: "Hello!", "200", dsb
  3. Booleans: true, false

selain tipe data di atas JavaScript juga mengenal tipe data null dan undefined. Null merupakan nilai dari sebuah variabel yang tidak diberi nilai saat didefenisikan. Sedangkan undefined merupakan hasil yang didapat dari proses berikut:
  1. Nilai dari pemanggilan variabel yang belum dideklarasikan
  2. Nilai dari pemanggilan elemn array yang tidak ada
  3. Nilai dari pemanggilan property object yang tidak ada
  4. Nilai dari pemanggilan function yang tidak mengembalikan nilai
  5. Nilai dari parameter function yang tidak memiliki argument


Operator

Operator merupakan sebuah simbol atau kata yang digunakan untuk melakukan operasi terhadap suatu atau lebih data atau variabel sehingga menghasilkan data baru. Variabel atau data yang dioperasikan disebut dengan operand. JavaScript mengenal beberapa jenis operator, yaitu sebagai berikut:

1. Operator Aritmatika

Operator aritmatika merupakan operator yang digunakan untuk melakukan perhitungan data numerik. Daftar operator aritmatika dan cara penggunaannya dilihat pada tabel berikut:

Operator Nama Operator Contoh
+ Penjumlahan 5 + 3
- Pengurangan 5 - 3
* Perkalian 5 * 3
/ Pembagian Misal A=10 dan B=4, maka A / B = 2.5
% Modulus Misal A=10 dan B=4, maka A % B = 2
** Pangkat Misal A=2 dan B=3, maka A**B = 8
++ Pengincrement Misal A=10, maka A++ menghasilkan 11
-- Pengdecrement Misal A=10, maka A-- menghasilkan 9

2. Operator Perbandingan

Operator perbandingan digunakan untuk membandingkan dua variabel atau data. Hasil perbandingan tersebut akan menghasilkan nilai boolean (True atau False). Daftar operator perbandingan dan penjelasannya dapat dilihat pada tabel berikut:

Operator Penjelasan Contoh
== Sama dengan Misal A=10 dan B=10, maka A==B = True
!= Tidak sama dengan Misal A=10 dan B=4, maka A!=B = True
> Lebih besar dari Misal A=10 dan B=4, maka A > B = True
< Kurang dari Misal A=4 dan B=10, maka A < B = True
>= Lebih besar atau sama dengan Misal A=10 dan B=10, maka A >= B = True
<= Kurang dari atau sama dengan Misal A=4 dan B=10, maka A <= B = True

3. Operator Logika

Operator logika digunakan untuk megoperasikan data boolean. Hasil perbandingan tersebut akan menghasilkan nilai true atau false. Daftar operator logika dan penjelasannya dapat di lihat pada tabel berikut:

Operator Penjelasan
&& Menghasilkan nailai true jika kedua operand bernilai true (logika And). Misal A = true dan B = false, maka (A && B) menghasilkan false.
|| Menghasilkan nilai true jika salah satu atau kedua operand bernilai true (logika OR). Misal A = true dan B = false, maka (A || B) menghasilkan true.
! Membalikkan nilai sebuah variabel (logika NOT). Misal A = true, maka (!A) menghasilkan false.

Dan masih ada operator Bitwise, Assignment, dan Kondisi, tetapi ini akan di pelajari kedepannya.
