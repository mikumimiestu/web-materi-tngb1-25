Table

Tabel adalah salah satu elemen HTML yang digunakan untuk menampilkan data dalam bentuk baris dan kolom. Tabel terdiri dari beberapa elemen, yaitu <table> sebagai elemen utama, <tr> sebagai elemen baris, <th> sebagai elemen header, dan <td> sebagai elemen data. Elemen <th> dan <td> harus berada di dalam elemen <tr>. Elemen <th> digunakan untuk menampilkan header, sedangkan <td> digunakan untuk menampilkan data.

Untuk contohnya adalah sebagai berikut:

<table>

<tr>

<th>Header 1</th>

<th>Header 2</th>

</tr>

<tr>

<td>Data 1</td>

<td>Data 2</td>

</tr>

</table>

Hasilnya sebagai berikut:

Header 1 Header 2
Data 1 Data 2

Dalam contoh di atas, kita dapat melihat struktur dasar HTML yang terdiri dari <table>, <tr>, <th>, dan <td>.


Tabel memiliki beberapa atribut yang dapat digunakan untuk mengatur tampilan tabel, seperti border untuk menampilkan garis batas, cellpadding untuk menambahkan jarak antara teks dan batas sel, cellspacing untuk menambahkan jarak antara sel, width untuk menentukan lebar tabel, dan height untuk menentukan tinggi tabel.


Lalu bagaimana jika kita ingin menambahkan style pada tabel tersebut? Kita bisa menggunakan tailwindcss untuk menambahkan style pada tabel tersebut. Berikut adalah contoh penggunaan tailwindcss pada tabel:

<div class="overflow-hidden rounded-lg border border-gray-300">

<table class="table-auto w-full">

<thead>

<tr class="bg-gray-200">

<th class="px-4 py-2 text-gray-700">Header 1</th>

<th class="px-4 py-2 text-gray-700">Header 2</th>

</tr>

</thead>

<tbody>

<tr>

<td class="border px-4 py-2">Data 1</td>

<td class="border px-4 py-2">Data 2</td>

</tr>

</tbody>

</table>

</div>

Hasilnya sebagai berikut:

Header 1 Header 2
Data 1 Data 2

Dalam contoh di atas, kita menggunakan class overflow-hidden untuk mengatur overflow tabel, rounded-lg untuk membuat sudut tabel menjadi rounded, dan border border-gray-300 untuk menambahkan border pada tabel. Selain itu, kita juga menggunakan class table-auto untuk mengatur lebar tabel secara otomatis dan w-full untuk membuat tabel menjadi full width. Untuk mengatur warna background header, kita menggunakan class bg-gray-200. Untuk mengatur warna background sel, kita menggunakan class border dan px-4 py-2 untuk mengatur padding pada sel. Untuk mengatur warna teks, kita menggunakan class text-gray-700 pada header.
