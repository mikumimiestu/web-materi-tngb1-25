Statement Control

Statement control pada bahasa pemrograman digunakan untuk menjalankan suatu program sesuai dengan kondisi tertentu. Ada berbagai jenis statement control pada JAvaScript yang dapat digunakan sesuai kebutuhan.

1. Percabangan dengan if

Percabangan ini digunakan untuk menjalankan suatu program jika kondisi terpenuhi (true) dan jika kondisi tidak terpenuhi tidak ada alternatif lain untuk dijalankan. Penggunaan if dapat di tulis dengan format sebagai berikut: 

                    
                        if (kondisi) pernyataan;

Jika pernyataan lebih dari satu baris, cara di atas tidak dapat digunakan, tetapi harus menggunakan format berikut: 

                    
                        if (kondisi) {
                            pernyataan baris pertama;
                            pernyataan baris kedua;
                        }

Berikut ini contoh percabangan menggunakan if pada JavaScript: 

                    
                        var umur = 20;
                        var dewasa;

                        if(umur > 17) dewasa = true;
                        if(umur > 20) document.write("Anda sudah boleh menikah");

                        if(dewasa) {
                            document.write("Anda sudah bisa buat KTP");
                            document.write("Anda sudah bisa buat SIM");
                        }

2. Percabangan dengan if ...else...

Percabangan ini digunakan untuk menjalankan program jika kondisi terpenuhi. Jika kondisi tidak terpenuhi, ada alternatif lain untuk dijalankan. Penggunaan if...else dapat di tulis dengan format sebagai berikut: 

                    
                        if (kondisi) pernyataan jika terpenuhi;
                        else pernyataan jika tidak terpenuhi;

Jika pernyataan lebih dari satu baris: 

                    
                        if (kondisi) {
                            pernyataan jika terpenuhi;
                        }else {
                            pernyataan jika tidak terpenuhi;
                        }

Berikut contoh penggunaannya: 

                    
                        var angka = 20;
                        if(angka % 2 == 0) document.write("Bilangan genap");

                        else document.write("Bilangan ganjil");
