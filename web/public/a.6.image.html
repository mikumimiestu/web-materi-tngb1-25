Image

Gambar adalah elemen HTML yang digunakan untuk menampilkan gambar. Elemen <img> digunakan untuk menampilkan gambar. Elemen ini memiliki atribut src yang digunakan untuk menentukan URL gambar. Elemen ini juga memiliki atribut alt yang digunakan untuk menentukan teks alternatif gambar. Teks alternatif ini akan muncul jika gambar tidak dapat ditampilkan.

Untuk contohnya adalah sebagai berikut:

<img src="https://dummyimage.com/1280x800/d4d4d4/171717" alt="Placeholder">

Dalam contoh di atas, kita dapat melihat struktur dasar HTML yang terdiri dari <img>.


Selain itu, kita juga dapat menambahkan atribut width dan height untuk menentukan lebar dan tinggi gambar. Atribut ini dapat diisi dengan nilai dalam satuan piksel atau persen. Jika hanya salah satu atribut yang diisi, maka gambar akan menyesuaikan ukuran yang diisi. Jika kedua atribut diisi, maka gambar akan diubah proporsinya. Jika tidak ada atribut yang diisi, maka gambar akan menyesuaikan ukuran aslinya.

Untuk gambar yang berupa file lokal, kita dapat menggunakan src yang mengarah ke path file gambar yang ingin kita tampilkan. Contoh sebagai berikut:

<img src="gambar.jpg" alt="Gambar">
