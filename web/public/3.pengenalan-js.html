Pemrograman Web

JavaScript

Pengertian

JavaScript merupakan bahasa pemrograman web yang pemrosesannya dilakukan di sisi client. Karena berjalan di sisi client, JavaScript dapat dijalankan hanya di browser. Berbeda dengan PHP yang bekerja di sisi server, untuk menjalankan script JavaScript tidak memerlukan refresh pada browser.

JavaScript biasanya di jalankan ketika ada event tertentu yang terjadi pada halaman web. Baik event yang dilakukan oleh user, maupun event yang terjadi karena adanya perubahan pada halaman website.

JavaScript mulai populer semenjak adanya jQuery sebagai salah satu library yang memudahkan pengguna JavaScript. Namun seiring munculnya library baru dan framework baru dengan berbagai kelebihannya, popularitaas jQuery mulai tergeser.

Meletakkan script JavaScript

Ada berbagai cara meletakkan code JavaScript pada dokumen HTML, namun biasanya disesuaikan dengan kebutuhan atau keadaan. Berikut berbagai cara meletakkan kode JavaScript.

1. Menggunakan Event Handler (Inline)

Cara ini biasanya digunakan jika kode JavaScript hanya sedikit atau sekedar memanggil function. Cara meletakkan kode JavaScript melalui cara ini yaitu dengan meletakkan kode setelah attribut event pada tag HTML. Contoh script berikut akan menampilkan kotak pesan ketika dokumen baru dibuka dan ketika di klik. 
                
                    <button onclick="alert('Halo, saya adalah pesan dari JavaScript!');
                    alert('Halo, saya adalah pesan dari JavaScript!')">Klik Saya

2. Menggunakan Tag Script (Internal)

Caranya yaitu dengan meletakkan kode JavaScript di dalam tag script. Tag ini dapat di letakkan di dalam tag head maupun body. Cara ini biasanya digunakan jika code JavaScript berisi beberapa baris tetapi tidak terlalu panjang. Biasanya menggunakan atribut type="text/javacript" untuk membedakan dengan script dari bahasa lain, tetapi tidak wajib. 
                
                    <script type="text/javascript">
                    alert('Halo, saya adalah pesan dari JavaScript!');
                    </script>

3. Menggunakan Tag Script src (External)

Caranya yaitu dengan memisahkan code JavaScript dengan code HTML. Code JavaScript di simpan dengan file tersendiri dengan format .js. File tersebut kemudian di panggil melalui atribut src pada tag script. Cara ini biasanya digunakan jika code JavaScript panjang dan kompleks. 
                
                    <script src="script.js"></script>
