Struktur Dasar HTML

Dalam membangun sebuah website di perlukan struktur dasar HTML yang jadi pembangun dasar dari sebuah website. Berikut adalah contohnya:

<!DOCTYPE html>

<html lang="en">

<head>

<meta charset="UTF-8">

<meta name="viewport" content="width=device-width, initial-scale=1.0">

<title>Pemrograman Web</title>

</head>

<body>

<h1>Hello World</h1>

<p>Ini adalah contoh paragraf</p>

</body>

</html>

Dalam contoh di atas, kita dapat melihat struktur dasar HTML yang terdiri dari <!DOCTYPE html>, <html>, <head>, dan <body>.

Pada code di atas hanyalah code html biasa, sekarang coba kita stayle dengan tailwind dan cdn sebagai berikut:

<link rel="stylesheet" href="https://cdn.tailwindcss.com">
